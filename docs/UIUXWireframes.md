# UI/UX Design Specifications

## Design Principles
- **Clean & Modern**: Minimalist design with focus on content
- **Brand Colors**: Navy Blue (#000046) and Emerald Green (#45cf7b)
- **Typography**: Inter for UI, IBM Plex Sans Arabic for Arabic
- **Spacing**: 8px grid system
- **Border Radius**: 12px for cards, 8px for inputs, 6px for buttons

## Component Design

### Input Fields
- Large, accessible touch targets (min 44x44px)
- Clear focus states (emerald green outline)
- Inline validation messages
- Character counters for text areas
- Helpful placeholders

### Buttons
- Primary: Emerald gradient with shadow
- Secondary: Navy blue outline
- Hover states with subtle animation
- Loading states with spinners
- Disabled states clearly indicated

### Cards
- White background with subtle shadow
- 12px border radius
- 24px padding
- Hover lift effect
- Focus visible for accessibility

### Feedback Elements
- Success: Emerald green with checkmark
- Error: Soft red with explanation
- Info: Navy blue with icon
- Warning: Amber with caution icon
- Always dismissible

### Mobile Responsive
- Stack layout below 768px
- Touch-optimized controls
- Swipeable panels
- Bottom sheet for results
- Floating action button
